# FlowMage

The missing flow/graph based GUI for image editing.
Powered by [Godot] and [FlatCV].

[Godot]: https://godotengine.org
[FlatCV]: https://github.com/ad-si/FlatCV

![Screenshot of the app](screenshots/2026-04-18t1221_app.png)
